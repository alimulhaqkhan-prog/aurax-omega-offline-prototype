# AURA-X Ω — Offline Emotional Continuity Prototype (Single-File)

یہ ریپو **AURA-X Ω** کا آف لائن پروٹوٹائپ ہے (Single HTML file) جس میں:
- TM/BM تصور کے مطابق چیٹ + ہسٹری
- BM Recall
- Dream Mode / Screensaver
- Multi-file attach (images/videos/audio/pdf/docs) → ایک ہی BM prompt میں
- LocalStorage based persistence (local-only)

## 1) Quick Start (Local)
1. اس repo کو download/clone کریں
2. `index.html` کو browser میں open کریں (Chrome recommended)

> نوٹ: کچھ features (مثلاً mic/camera permission) صرف HTTPS یا localhost پر بہتر چلتے ہیں۔

## 2) GitHub Pages (Online Demo)
1. Repo → **Settings** → **Pages**
2. **Source**: Deploy from a branch
3. Branch: `main` / folder: `/ (root)`
4. Save → چند لمحوں بعد URL بن جائے گا

## 3) Data Storage
یہ prototype **صرف local storage** استعمال کرتا ہے:
- BM / Intelligence / History / User Identity وغیرہ
- کوئی server پر upload نہیں ہوتا جب تک آپ خود LLM settings میں API نہ لگائیں۔
